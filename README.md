# RTU Student Project Archive

Welcome to the RTU Student Project Archive!

This repository contains programming projects submitted by students.

## Purpose
- Build student portfolios
- Encourage collaboration
- Showcase IT student work

## Submission Process
Students submit their projects through Google Forms.
Approved projects are uploaded here.

## Rules
- All submissions must be original.
- No plagiarized or harmful content.
- Proper credit will always be given.
